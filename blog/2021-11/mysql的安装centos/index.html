MySQL的安装（Centos）
  • Nov 12, 2021

1. 下载MySQL版本

点击这里选择版本下载，一般来说centos都是用5.7的版本

下载完之后这个包需要上传到服务器上，可以用的软件有XFTP，直接放到/usr/local/下面

2. 安装

  1. 用“tar -zxvf”命令解压上传的包

  2. 用“mv 解压完的目录 改名的目录”命令改一下目录名字

  3. 创建用户和组并修改权限：

    groupadd mysql#mysql 是组名
useradd -r -g mysql mysql#第二个mysql是用户名
mkdir -p /data/mysql #创建数据存储目录
chown mysql:mysql -R /data/mysql #修改权限

  4. 修改配置文件my.cnf（在/etc/my.cnf中，如果没有就自己创建）

    [client]
socket=/usr/local/mysql/mysql.sock
   
[mysqld]
basedir=/usr/local/mysql
datadir=/data/mysql
socket=/usr/local/mysql/mysql.sock
port=3306
user=mysql
character_set_server=utf8mb4
# Disabling symbolic-links is recommended to prevent assorted security risks
symbolic-links=0
# Settings user and group are ignored when systemd is used.
# If you need to run mysqld under a different user or group,
# customize your systemd unit file for mariadb according to the
# instructions in http://fedoraproject.org/wiki/Systemd
   
[mysqld_safe]
log-error=/data/mysql/mysql.err
pid-file=/data/mysql/mysql.pid

    （注意：其中的basedir、datadir、socket等路径必须对应你安装MySQL的路径，另外mysql.sock这个文件是用于服务器本地客户端和MySQLserver端通信的文件，也就是socket的方式，所以两个地方的路径是一致的）

  5. 开始初始化数据库

    ./mysqld --defaults-file=/etc/my.cnf --basedir=/usr/local/soft/mysql/ --datadir=/data/mysql/ --user=mysql --initialize

    对应的路径都需要修改正确，就跟cnf文件中的一致，执行完会出现一下内容

  6. 开启MySQL服务

    先将mysql.server 放置到/etc/init.d/mysql中

    cp /usr/local//mysql/support-files/mysql.server /etc/init.d/mysql

    然后用下面的命令就可以启动了

    service mysql start

3. 登录

  1. 登录MySQL

    进入MySQL的/bin/目录中，然后用以下命令登入：

    ./mysql -uroot -p临时密码

    然后执行下面三行命令：

    SET PASSWORD = PASSWORD(‘123456’);
    ALTER USER ‘root’@‘localhost’ PASSWORD EXPIRE NEVER;
    FLUSH PRIVILEGES;

    这也就将密码改成了123456了。

  2. 开放端口外部连接

    执行下面的命令完成操作：

    use mysql; update user set host = ‘%’ where user = ‘root’; flush privileges;

  3. 快捷登录

    用以下命令使用软连接：

    ln -s /usr/local/mysql/bin/mysql /usr/bin

    然后就可以用“mysql -uroot -p”命令直接登录而不用进入/bin目录下。
